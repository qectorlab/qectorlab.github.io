## QECTOR Decoder

****QECTOR** Decoder** is the available core decoder library.

It provides a quantum error correction decoder stack for research, benchmarking, and commercial **QEC** workflow evaluation.

Core features:

- Rust core with Python bindings
- PyO3 integration
- Stim Detector Error Model support
- Sinter integration
- PyMatching-compatible workflows
- Exact weighted **MWPM** mode
- Belief-matching accuracy mode
- BP-**OSD** / **LDPC** / qLDPC support
- **CPU** batch decoding
- **CUDA** batch decoding
- OpenCL batch decoding
- Reproducible benchmark artifacts
- Artifact **SHA**-**256** hashes
- Hot-path and cold-path timing
- p50 / p90 / p95 / p99 latency reporting
- Native memory profiling
- Benchmark reproduction scripts
- Commercial licensing support
