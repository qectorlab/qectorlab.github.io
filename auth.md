# Auth.md

QECTOR Store agent authentication and registration metadata.

## Service

QECTOR Store is a public documentation, discovery, and licensing website for QECTOR Decoder 3.

Canonical site: https://qector.store

## Public access

There is no public protected API and no automated OAuth token issuance for the public static website at this time.

Agents can use these public resources without authentication:

- Website: https://qector.store/
- Install guide: https://qector.store/installer.html
- Documentation: https://qector.store/docs.html
- Benchmark notes: https://qector.store/benchmarks.html
- LLM summary: https://qector.store/llms.txt
- API catalog: https://qector.store/.well-known/api-catalog
- OpenAPI discovery document: https://qector.store/openapi.json
- PyPI package: https://pypi.org/project/qector-decoder-v3/
- GitHub repository: https://github.com/GuillaumeLessard/qector-decoder
- DOI: https://doi.org/10.5281/zenodo.20825980

## Agent registration

Automated registration is not available for this public static site.

Human-reviewed registration, licensing, source review, OEM, enterprise, SaaS, redistribution, or protected integration requests should be sent through:

https://qector.store/contact.html

## Supported identity types

- Individual researcher
- Academic lab
- Startup
- Commercial R&D team
- Enterprise buyer
- OEM partner

## Credential types

No bearer token, OAuth client credential, API key, or signed agent credential is issued automatically from the public website.

## Revocation and security contact

Security and access questions should be submitted through:

https://qector.store/contact.html
