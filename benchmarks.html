Validated Claims — v0.5.3

What we have actually measured

Exact MWPM Parity

QECTOR BlossomDecoder achieves bit-identical logical error rates with PyMatching through distance 15 on rotated surface codes under circuit-level depolarizing noise (Wilson 95% CI overlap).

Belief-Matching Improvement

BeliefMatching delivers up to 35.7% lower logical error rate compared to PyMatching on correlated circuit-level noise at distance 5.

GPU Batch Throughput

OpenCLBatchDecoder sustains 14.6 million decodes per second on large batches with full bit-identical results across CPU and GPU paths.

BP-OSD / qLDPC Support

BpOsdDecoder successfully decodes hypergraph product and bivariate bicycle codes within ~10% LER of reference ldpc package implementations.

Reproducibility Standard

Every benchmark includes

SHA-256 Hashed Artifacts

All result files, logs, and reports are hashed. Hashes are included in every published benchmark.

Full Environment Snapshot

CPU model, GPU driver, CUDA/OpenCL version, Python/Rust versions, and exact package versions are recorded.

Git Commit Stamp

Every benchmark run is tied to a specific git commit so results can be exactly reproduced later.
