✓ Exact MWPM LER parity with PyMatching through d=15 (overlapping Wilson 95% CI)
✓ Belief-Matching achieves up to 35.7% lower LER than PyMatching on correlated circuit-level noise
✓ CPU/GPU batch decoding is bit-identical across all tested distances and batch sizes
✓ BP-OSD decodes qLDPC codes (BB, HGP) within ~10% LER of reference ldpc package
✓ All benchmark artifacts are SHA-256 hashed and traceable to a specific build commit
