Executive Summary

QECTOR v3 is a poly-algorithmic decoding suite featuring a high-performance Rust core (Blossom MWPM, Sparse Blossom, Union-Find, BP-OSD) with full Python ecosystem integration. It delivers exact LER parity with PyMatching on weighted MWPM up to distance 15, while its belief-matching mode consistently achieves lower logical error rates than standard MWPM on circuit-level noise.

Honest Limitations

PyMatching remains the CPU latency leader (typically 3x–24x faster). Belief-matching is the accuracy mode and is significantly slower per shot. BP-OSD is competitive but trails specialized ldpc implementations by ~10% on certain qLDPC codes. Union-Find is a fast approximate path.
