QECTOR Decoder v3

Source-Available Quantum Error Correction Decoder

Version 0.5.3 — Rust core with Python API. Exact MWPM, Belief-Matching, BP-OSD, GPU batch decoding.

Quick Start

Copy & paste these three commands

1. Install QECTOR Decoder v3 
pip install qector-decoder-v3
2. Verify the installation 
python -c "from qector_decoder_v3 import UnionFindDecoder, BlossomDecoder; print('QECTOR OK')"
3. Run your first decode (repetition code) 
import numpy as np
from qector_decoder_v3 import generate_repetition_code_checks, UnionFindDecoder

checks, n_qubits = generate_repetition_code_checks(5)
syndrome = np.zeros(len(checks), dtype=np.uint8)
syndrome[0] = 1

decoder = UnionFindDecoder(checks, n_qubits)
correction = decoder.decode(syndrome)
print(correction)
Installation Options

Recommended installs

pip install qector-decoder-v3
pip install "qector-decoder-v3[stim]"     # + Stim, Sinter, PyMatching
pip install "qector-decoder-v3[all]"      # Full environment
