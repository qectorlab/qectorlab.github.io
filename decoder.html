QECTOR Decoder v3

Source-Available Quantum Error Correction Decoder

Version 0.5.3 — Rust core with Python API. Exact MWPM, Belief-Matching, BP-OSD, GPU batch decoding, and reproducible artifacts.

Install from PyPI v0.5.3 View Benchmarks Commercial License
Installation

pip install qector-decoder-v3

pip install qector-decoder-v3
pip install "qector-decoder-v3[stim]"   # Stim + Sinter + PyMatching
pip install "qector-decoder-v3[all]"    # Full environment
Decoders Included (v0.5.3)

Choose the right backend for your workload

Matching Decoders

  • BlossomDecoder — Exact weighted MWPM (PyMatching parity)
  • SparseBlossomDecoder — Near-optimal, faster on large graphs
  • UnionFindDecoder + FastUnionFindDecoder — Lowest latency

Accuracy & LDPC

  • BeliefMatching — Up to 35.7% lower LER on correlated noise
  • BpOsdDecoder — BP-OSD for qLDPC codes (BB, HGP, etc.)

Batch & GPU

  • CUDABatchDecoder + OpenCLBatchDecoder — 14.6M dec/s
  • AutoDecoder — Automatically chooses best backend

Specialized

  • StreamingDecoder, SlidingWindow, HybridDecoder
  • LookupTableDecoder, PredecodedDecoder
