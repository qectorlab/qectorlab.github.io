QECTOR Decoder

The core QEC decoder library for researchers, benchmarking workflows, and commercial QEC evaluation. Rust core with Python bindings via PyO3.

  • Exact weighted MWPM (Blossom)
  • Belief-matching accuracy mode — up to 35.7% lower LER
  • BP-OSD for LDPC / qLDPC codes
  • CUDA + OpenCL batch decoding (14.6M dec/s)
  • Stim DEM loading, Sinter plug-in
  • Drop-in PyMatching-compatible API
  • Reproducible artifacts with SHA-256 hashes
  • 395+ tests, including brute-force optimality checks

QECTOR Workbench

A local fullstack app for loading circuits, running decoder comparisons, exporting artifacts, and generating reproducible benchmark reports — no cloud required.

  • Load .stim circuits and .dem files
  • Run QECTOR vs PyMatching comparisons
  • Generate PDF benchmark reports from real data
  • Export CSV / JSON artifacts with environment snapshot
  • Verify SHA-256 artifact hashes
  • CPU / CUDA / OpenCL backend diagnostics
  • Visualize LER, latency, threshold, and memory
  • Job queue with cancel support

PyMatching remains the latency leader for single-shot exact MWPM — QECTOR does not claim otherwise. QECTOR's validated advantages are: logical accuracy parity through d=11, belief-matching LER reduction (up to 35.7% at d=5), BP-OSD for qLDPC codes matching can't handle, GPU batch throughput (14.6M dec/s), and commercial workflow packaging with reproducible artifacts.
