QEC decoding for teams that need reproducible proof, not just a fast baseline.

QECTOR Decoder v3 is a source-available Rust/Python quantum error correction decoder platform by Guillaume Lessard / iD01t Productions. It supports PyMatching-compatible validation, belief-matching experiments, BP-OSD / LDPC workflows, CPU/GPU batch paths, and artifact-backed benchmark evidence.

Install from PyPIReview EvidenceView DOI

Free for personal, academic, educational, and non-commercial research. Commercial evaluation starts at $1,500 / 90 days.

Public install
pip install qector-decoder-v3
python -c "from qector_decoder_v3 import UnionFindDecoder, BlossomDecoder; print('QECTOR OK')"

Use the PyPI wheel for normal installs. Source builds are for licensed/source-review workflows.

Market position

Why QECTOR exists beside PyMatching and Stim

Not a universal fastest-decoder claim.
PyMatching remains the latency leader for exact MWPM on common tested workloads. QECTOR should not be sold as a universal PyMatching replacement.

The paid value

QECTOR is for labs, startups, and commercial R&D teams that need one legally clear platform for multi-decoder experiments, BP-OSD/LDPC workflows, GPU batch validation, reproducible artifacts, source access, and benchmark reports.
