QECTOR Decoder v3

The core high-performance decoder library. Rust core with clean Python API. PyMatching-compatible.

  • Exact weighted MWPM (BlossomDecoder)
  • Belief-Matching — up to 35.7% lower LER
  • BP-OSD for qLDPC / LDPC codes
  • CUDA + OpenCL batch decoding (14.6M dec/s)
  • Stim DEM loading + Sinter plug-in
  • Drop-in PyMatching-compatible API
  • SHA-256 hashed reproducible artifacts
  • 414+ tests with brute-force validation

QECTOR Workbench

Local fullstack application for circuit loading, decoder comparison, artifact export, and reproducible technical reports.

  • Load .stim circuits and .dem files
  • Run QECTOR vs PyMatching head-to-head
  • Generate PDF benchmark reports
  • Export CSV/JSON with SHA-256 hashes
  • CPU / CUDA / OpenCL diagnostics
  • Visual LER, latency, threshold curves
  • Job queue with cancel support
