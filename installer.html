Installation

Install QECTOR Decoder v3

Simple pip installation with optional extras for Stim, Sinter, and full benchmarking environments.

Basic Installation

pip install qector-decoder-v3

Recommended Installs

# With Stim + Sinter + PyMatching compatibility
pip install "qector-decoder-v3[stim]"

# Full environment (recommended for serious use)
pip install "qector-decoder-v3[all]"

Verify Installation

python -c "from qector_decoder_v3 import UnionFindDecoder, BlossomDecoder; print('QECTOR OK')"
