What it fixes

Real installer coverage

Missing Rust

Installs Rust with rustup and detects Cargo paths after setup.

Wrong Python

Detects bundled wheel tags like cp311 and recreates the virtual environment with a compatible interpreter when available.

Thin checkout

Searches the parent release folder and Downloads for the real source payload and repairs the clone.

Git Bash linker issue

Removes Git Bash linker path conflicts before Rust source builds.

Windows source build

When a native build is required

If no matching wheel is available and QECTOR must compile from source on Windows, install Visual Studio Build Tools with the C++ build tools workload. The MSVC Rust target needs Microsoft linker support.
