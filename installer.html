Official PyPI installv0.5.1Wheel first

Install QECTOR Decoder 3 from PyPI.

The normal public path is the PyPI wheel. Use the same Python environment for install and verification.

PyPI commandVerify installOpen PyPI
Terminal
pip install qector-decoder-v3
python -c "from qector_decoder_v3 import UnionFindDecoder, BlossomDecoder; print('QECTOR OK')"
Recommended command

Use the public PyPI package.

This installs qector-decoder-v3. On Windows, use the pip attached to the Python you actually run with python.

pip install qector-decoder-v3

If you maintain multiple Python installs, check pip --version, python --version, and py -0p.

Verification

Expected successful result.

python -c "from qector_decoder_v3 import UnionFindDecoder, BlossomDecoder; print('QECTOR OK')"

Expected output:

QECTOR OK
