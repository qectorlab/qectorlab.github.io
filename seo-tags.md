# QECTOR Decoder 3 SEO tags

QECTOR Decoder 3 is a Rust/Python quantum error correction decoder package for reproducible QEC research, MWPM, Union-Find, BP-OSD, LDPC/qLDPC, Stim/PyMatching validation, CUDA batch decoding, GPU acceleration, and high-throughput scientific computing workflows.
