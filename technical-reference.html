Technical reference

Decoder scope, package scope, and claim scope.

Use this page as a technical positioning reference before comparing QECTOR with PyMatching, Stim, Sinter, BP-OSD, LDPC, or qLDPC workflows.

Package

Public install name: qector-decoder-v3. Standard CPython wheels are the normal user path.

Decoder scope

UnionFindDecoder, BlossomDecoder, and selected advanced workflows are documented in the GitHub README.

Evidence scope

Benchmark claims are scoped to public artifacts and local reproduction.

Technical boundary.
Do not turn scoped benchmark data into universal product claims. Reproduce locally for the target workload and environment.
