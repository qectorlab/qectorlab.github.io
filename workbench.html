QECTOR Workbench

The planned local validation workstation for commercial R&D teams that need reproducible QEC reports, artifact hashes, environment snapshots, and clean QECTOR vs PyMatching comparison workflows.

Join Early AccessView Benchmark EvidenceView DOI
Why it matters

The differentiator is not another decoder button. It is auditable proof.

PyMatching and Stim are excellent tools. Workbench is designed around the gap they do not try to fill: packaging a full benchmark run into artifacts a lab, startup, investor, or customer can inspect and reproduce.

Artifacts

Export CSV and JSON outputs with run metadata, benchmark parameters, and decoder selections.

Hashes

Compute SHA-256 hashes so benchmark bundles can be checked and archived.

Reports

Generate PDF reports from real runs instead of marketing-only screenshots.

Local-first

Run sensitive workflows locally by default, without sending internal circuits to a cloud service.

Planned workflow

From circuit to reproducible report

Load

Import `.stim` circuits, `.dem` files, benchmark configurations, and artifact bundles.

Run

Execute QECTOR, PyMatching-compatible comparisons, belief-matching, BP-OSD experiments, and backend diagnostics.

Prove

Export CSV/JSON artifacts, SHA-256 hashes, environment snapshots, DOI references, and PDF reports from real benchmark data.

Features

Local QEC benchmark workflow

  • Load .stim circuits and .dem files
  • Run QECTOR vs PyMatching comparisons
  • Run belief-matching accuracy benchmarks
  • Run BP-OSD / LDPC experiments
  • Generate PDF benchmark reports
  • Export CSV / JSON artifacts
  • Capture environment and dependency versions
  • Verify SHA-256 artifact hashes
  • Reference PyPI package version and Zenodo DOI
  • Visualize LER, latency, threshold, memory, and scaling
  • Job queue with cancel support
  • Local-first workflow, no cloud required by default
Roadmap

Release path

01

Benchmark report skeleton

PDF/CSV/JSON export format, environment capture, artifact hash manifest, and report template.

02

Headless backend

Benchmark execution, cancellation, export paths, backend detection, and reproducibility bundle generation.

03

Local interface

Project dashboard, circuit loader, decoder selector, charts, artifact browser, and report export.

04

Commercial alpha

Private early-access build for serious commercial evaluators and labs under license terms.

05

Public release

Release only after a working build, docs, examples, support flow, and validated artifact exports exist.
