QECTOR Workbench

A local fullstack QEC benchmarking and validation app for loading circuits, running decoder comparisons, exporting artifacts, verifying hashes, and generating reproducible technical reports from one interface.

Join Early AccessView DecoderRequest Commercial Demo
Purpose

From decoder library to validation workstation

The decoder library already provides the engine and evidence pipeline. Workbench packages the workflow into a local app: load circuits, run decoders, compare results, capture environments, export artifacts, and generate reproducible reports without sending sensitive work to a cloud service.

Load

Import Stim circuits, Detector Error Models, benchmark configurations, and artifact bundles.

Run

Execute QECTOR, PyMatching-compatible comparisons, belief-matching, BP-OSD experiments, and backend diagnostics.

Prove

Export CSV/JSON artifacts, SHA-256 hashes, environment snapshots, and PDF reports from real benchmark data.

Planned Features

Local QEC benchmark workflow

  • Load .stim circuits and .dem files
  • Run QECTOR vs PyMatching comparisons
  • Run belief-matching accuracy benchmarks
  • Run BP-OSD / LDPC experiments
  • Generate PDF benchmark reports
  • Export CSV / JSON artifacts
  • Capture environment and dependency versions
  • Verify SHA-256 artifact hashes
  • Run CPU / CUDA / OpenCL diagnostics
  • Visualize LER, latency, threshold, memory, and scaling
  • Job queue with cancel support
  • Local-first workflow, no cloud required by default
